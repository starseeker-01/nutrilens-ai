# NutriLens AI 

![NutriLens Logo](assets/nutri%20logo.png)

**AI Nutrition Assistant with Food Image Analysis**

##  Features
-  **Food Image Recognition** - Upload food photos for instant analysis
-  **Nutritional Breakdown** - Get detailed calorie, protein, carb, and fat information
-  **Health Insights** - Personalized recommendations based on your meals
-  **Progress Tracking** - Monitor your nutritional journey over time

##  Technologies Used
- **AI/ML**: TensorFlow/PyTorch, Computer Vision
- **Backend**: Python, FastAPI/Flask
- **Frontend**: Streamlit
- **Database**: MongoDB

##  Installation

```bash
# Clone the repository
git clone https://github.com/starseeker-01/nutrilens-ai.git
cd nutrilens-ai

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
